## 사용한 라이브러리 및 프레임워크
- 백엔드(Python)
  - DjangoRestFramework
  - ADRF(Async Django REST framework) [(Github)](https://github.com/em1208/adrf)
  - openai
  - asyncpg(PostgreSQL 비동기 드라이버)
  - uvicorn(비동기 처리를 위한 asgi)
  - decouple(ChatGpt API키 .env 파일로 분리)
- 프론트엔드(JavaScript)
  - Next.js
  - React
  - diff-match-patch
